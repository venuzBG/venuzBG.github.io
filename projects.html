Proyecto Final - Guitarra Virtual

Este proyecto fue desarrollado como trabajo final de la asignatura Programación II. Su objetivo fue integrar el uso de una base de datos SQL con una interfaz gráfica para gestionar usuarios y permitir la creación de una aplicación interactiva: una guitarra virtual.

La aplicación permite al usuario ingresar al sistema, tocar notas musicales con el teclado, grabarlas y luego reproducirlas. Cada grabación se almacena y puede ser gestionada por el usuario.

Arquitectura del sistema

La aplicación se diseñó con una arquitectura en capas que separa la interfaz gráfica, la lógica de negocio y el manejo de la base de datos.

Diagrama de clases

Este diagrama representa las clases principales de la aplicación y sus relaciones. Permite visualizar cómo interactúan las entidades dentro del sistema.

Modelo Entidad-Relación (MER)

El modelo entidad-relación muestra la estructura de la base de datos utilizada para almacenar la información de usuarios y canciones creadas en la aplicación.
